## 遊戲規則 — SnackGame

核心設定
- 棋盤: 20 × 20 格
- 控制: 方向鍵 或 W/A/S/D 控制方向（不可直接反向）
- 開始/重玩: `Space`，暫停/繼續: `P`
- 畫面行為: 穿牆（從一邊出界會從對邊出現）

食物種類
- 普通 (紅色): 吃到 +1 分
- 黃金 (金色): 吃到 +3 分（出現機率較低）
- 毒物 (紫色): 吃到 -1 分，且會縮短蛇身（若分數為 0 則維持 0）

道具 (Power-ups)
- 加速 (`speed`): 在有限時間內遊戲速度提升
- 減速 (`slow`): 在有限時間內遊戲速度降低
- 無敵 (`invincible`): 在有限時間內觸碰自己不會死亡

連鎖機制（Combo）
- 在 `comboTimeout`（預設 2 秒）內連續吃到食物會增加連鎖倍率，倍率會疊加並影響此次得分。若超過時限未吃到新食物，連鎖歸零。

食物壽命
- 每個食物有壽命（預設 5 秒），若在壽命內未被吃掉會自動消失並重新產生一個新食物。

其他
- 分數無上限，遊戲以「撞到自己（非無敵時）」結束。遊戲結束後畫面會停止，按 `Space` 可重置並開始新局。
- 畫面右側有說明面板顯示食物與道具說明。

實作說明（開發者快速提示）
- 主要程式檔: `src/components/SnakeGame.vue`
- 啟動 (本機): `npm install` → `npm run dev`

待調整其他規則加入或調整參數（例如食物壽命、連鎖時限、道具種類）。
